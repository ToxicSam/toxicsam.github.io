Toon Shaders

Robert Qiu, Brandon Bizzarro, Sanwu Luo, Leon Wu

Summary

A collection of realtime toon / stylized shaders made for the end goal of use in a game engine.

Problem Description

We aim to create a collection of simple toon or stylized shaders, not because there is a dearth of them, but because we wish to learn how to create realtime shaders for personal enrichment and possible inclusion in future 3D interactive projects. Realtime shaders is more or less a new frontier for all of us, and creating stylized shaders allows us to express more creative aspects in this project.
